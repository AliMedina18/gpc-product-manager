# 01 - Setup

## Requisitos
- Node.js 18+
- PostgreSQL 13+
- npm

## Pasos iniciales

```bash
git clone <url>
cd gpc-product-manager
```

## Backend

```bash
cd backend
npm install
cp .env.example .env
```

Editar `backend/.env` con la URL de PostgreSQL y el secreto JWT.

## Frontend

```bash
cd frontend
npm install
```
