# 03 - Frontend

## Estructura

- `frontend/src/App.jsx`
- `frontend/src/components`
- `frontend/src/pages`
- `frontend/src/services`
- `frontend/src/hooks`
- `frontend/src/layouts`

## Comandos

```bash
cd frontend
npm install
npm run dev
npm run build
npm run preview
npm run lint
```

## Variables

- `VITE_API_URL` debe apuntar al backend local o remoto.
